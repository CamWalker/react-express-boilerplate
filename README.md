# Language
    ECMAScript2015 (ES6)
    
# Description
    This Project is Serverside rendering for react on express.

# Usage
#### 1. Install yarn and pm2
    npm install -g yarn pm2
#### 2. Rename config file and then edit the file.
    .envcpy to .env
#### 3. run server
    for development
      $ yarn start:dev
    for production (using pm2)
      $ yarn start:prod
